# Wallet

Budget tracking app for Gabriel's (me) family, built with [Next.js](https://nextjs.org/), [TailwindCSS](https://tailwindcss.com/) and [Prisma](https://www.prisma.io/).

## Scripts

To run:

```sh
$ npm run dev
```

To build:

```sh
$ npm run build
```
